1. Empathy_Map_Canvas.docx

Focuses on understanding the needs, behaviors, and pain points of stakeholders such as students, faculty, and administrators.

Says: “Managing student data manually is time-consuming.”
Thinks: “Is there a smarter way to handle these repetitive tasks?”
Does: Fills out admission forms manually, updates marks in Excel, etc.
Feels: Frustrated, anxious, bored, but curious about digital solutions.

Pain Points: Manual data entry, human error, paperwork overload.
Gains / Needs: A digital platform for error-free admission tracking and auto-calculated results.

Key Insight: Implementing ServiceNow can unify academic, administrative, and IT operations, reducing inefficiency and increasing user satisfaction.

2. Define_Problem_Statements.docx

Identifies real-world issues in educational institutions and reformulates them into actionable problem statements.

Core Problem:
Manual processes and disconnected systems lead to inefficiencies, delays, and poor visibility in service management.

Solution Goal:
Use ServiceNow to automate workflows and centralize communication between departments.

Defined Problem Statements (PS):

3. Brainstorming_Idea_Generation_Prioritization.docx

Documents the brainstorming and prioritization phase of the ideation process.

Selected Problem Statement:
“How might we automate student admissions and academic performance tracking in educational institutions using ServiceNow?”

Grouped Ideas:
- Admissions Automation: Auto-generated admin number, pre-filled forms
- Performance Tracking: Auto-calculation of totals and percentages
- Workflow & UX: Status-based flow, role-based access control
- Scalability: Notification integration, reporting features

Priority Matrix:
Idea Group	Priority	Reason
Admissions Automation	High	Core functionality
Performance Tracking	High	Improves accuracy
Workflow & UX	Medium	Enhances usability
Scalability	Low	Future expansion
Project Summary

This project aims to transform educational administration through ServiceNow automation.
It focuses on improving admission workflows, student performance tracking, and inter-departmental communication by replacing manual processes with digital solutions.

Tools & Platform

- ServiceNow (Low-code Platform)
- Microsoft Word / Docs (Documentation)
- Team Collaboration Tools (Virtual ideation sessions)

Expected Outcomes

- Centralized platform for academic and administrative services
- Automated workflows reducing time and errors
- Enhanced communication between staff and management
- Better transparency and satisfaction across stakeholders
